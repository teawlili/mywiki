换行
hexo下,直接回车键就可以换行.无需在行尾加上两个空格键再回车.

> 这段文字将会被高亮显示

```Python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
print 'Hello World! 
```

例如，如果你需要插入反斜杠，就连续输入两个反斜杠即可：\\=> \ 。

注：在内容中输入以上特殊符号的时候一定要注意转义，否则将导致内容显示不全，甚至排版混乱。

一个竖线: &#124;
两个竖线: &#124;&#124;

不断行空格 &nbsp;
半角空格 &ensp;
全角空格 &emsp;

例：这&nbsp;是&ensp;个&emsp;例子
